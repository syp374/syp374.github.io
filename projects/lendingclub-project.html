🎯 LendingClub Loan Default Classification Model

Project Duration: 2025.04 - 2025.04 (3 weeks)

👥 Team Size: 5 members

📌 Core Skills: Classification Modeling • Class Imbalance Handling • Statistical Analysis

📋 Project Overview:

Developed loan default classification models for US P2P lending platform LendingClub, improving model performance despite severe class imbalance and optimizing financial risk management.

⚙️ Technical Approach:

  • SMOTE Oversampling: Addressed severe class imbalance (performing:default = 9:1)
  • Ensemble Modeling (XGBoost, LightGBM): Built robust loan default classification framework
  • Feature Engineering: Selected high-predictive variables from 150+ columns and created additional metrics

📊 Key Achievements:

  • Achieved ROC-AUC of 0.89 (17% improvement over baseline)
  • 68% loan default detection rate with enhanced screening recommendations
  • 15-20% projected loss reduction through improved loan approval processes

🎯 Skills Acquired:

  • Machine Learning Performance Optimization: Feature engineering and hyperparameter tuning expertise
  • Financial Risk Modeling: Cost-based evaluation for practical model deployment
  • High-dimensional Data Processing: Core feature selection and optimization capabilities
