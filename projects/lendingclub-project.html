🎯 LendingClub 대출 미상환자 분류 모델

핵심 스킬: 머신러닝 분류 + 클래스 불균형 처리 + 통계 분석

📋 프로젝트 개요:

  • 미국 P2P 대출 플랫폼 LendingClub 데이터로 대출 미상환자 분류 모델 개발
  • 심각한 클래스 불균형(정상:부실 = 9:1) 문제 속 모델링 개선 및 금융 리스크 관리 최적화

⚙️ 기술적 접근:

  • Python(Scikit-learn, XGBoost, LightGBM), SMOTE 오버샘플링, 앙상블 모델링 적용
  • 150+ 변수 중 Feature Engineering을 통한 변수 선택, 비용 기반 평가 지표 설계, ROC Curve 분석 기반 최적 임계값 탐색

📊 주요 성과:

  • XGBoost + SMOTE 조합으로 ROC-AUC 0.89 달성(베이스라인 대비 17% 향상), 대출 미상환자 탐지율 68% 기록
  • 예상 손실 감소율 15-20% 달성으로 대출 심사 자동화 및 효율성 개선 방안 도출
