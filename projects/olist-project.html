🎯 Olist Brazilian E-commerce Data Analysis

Project Duration: 2025.05 - 2025.05 (3 weeks)

👥 Team Size: 5 members

📌 Core Skills: Tableau Dashboard • Time Series Analysis • Marketing Funnel Analysis

📋 Project Overview:

Enhanced seller experience and developed new seller acquisition strategies for Brazilian B2B e-commerce platform Olist through sales analysis, customer segmentation, and logistics optimization.

⚙️ Technical Approach:

  • Multi-method Outlier Detection: Applied Z-Score, IQR, and Isolation Forest techniques to identify abnormal transaction patterns
  • Complex Data Relationship Management: Handled many-to-many relationship duplications ensuring data integrity across multiple table merges
  • Marketing Funnel Analysis: Identified bottlenecks in customer conversion processes

📊 Key Achievements:

  • Built real-time KPI monitoring dashboard using Tableau
  • Developed platform competitiveness enhancement strategies across Sales, Quality, Logistics, and Marketing dimensions

🎯 Skills Acquired:

  • Data Quality Management: Experience ensuring data integrity in complex relational datasets
  • Business Intelligence: Real-time monitoring system development capabilities
  • Funnel Analysis: Data-driven insights for customer journey optimization
