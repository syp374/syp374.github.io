🎯 한국문학 번역 시장 데이터 기반 의사결정 지원 플랫폼

프로젝트 기간: 2025.06 ~ 2025.07 (1개월)

프로젝트 인원: 5명

핵심 스킬: LLM 기반 텍스트 분석 + 데이터 증강 + 군집 및 페르소나 분석 + Streamlit 대시보드

📋 프로젝트 개요:

  • 한국 소설의 글로벌 번역 시장 진출을 돕기 위한 데이터 기반 의사결정 지원 플랫폼 구축
  • 아마존 베스트셀러, IMDB 인기 지표 등 외부 데이터를 분석하여 해외 시장 성공 가능성이 높은 소설 식별

⚙️ 기술적 접근:

  • LLM(Gemini API) 활용: 소설 장르 및 마케팅 표현 자동 분류
  • SMOTified-GAN 및 TVAE: 데이터 불균형 문제 해결
  • CatBoost, 로지스틱 회귀 등 분류 모델 비교 및 교차 검증: 최종 예측 모델 선정 및 모델 성능 향상

📊 주요 성과:

  • F1-score 0.52 → 0.61 향상으로 예측 정확도 개선
  • FUZZY TOPSIS 기반 복합 평가지표 개발 및 번역 우선순위 선정
  • SHAP 분석을 통한 모델 해석 가능성 확보 및 주요 변수 영향력 시각화

🎯 획득 역량:

  • 텍스트 데이터 처리: LLM API 활용한 대량 텍스트 자동 분류 역량
  • 딥러닝을 활용한 데이터 증강: 데이터 불균형 해결 및 모델 최적화 경험
  • Streamlit 대시보드 앱 개발 및 배포: 실무진이 활용 가능한 의사결정 도구 설계 능력

🏆 수상 실적:
