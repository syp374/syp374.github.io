🎯 H&M 고객 충성도 데이터 분석

프로젝트 기간: 2025.03 - 2025.03 (2주)

👥 팀 규모: 5명

📌 핵심 스킬: 고객 세분화 • 충성 고객 정의 • EDA/선호도 분석

📋 프로젝트 개요:

RFM 기반 충성 고객 정의를 통해 H&M 구매 데이터를 분석하고, 충성 고객과 일반 고객의 상품 선호 차이를 분석/검증했습니다.

⚙️ 기술적 접근:

  • RFM 분석: 고객 충성도 지표를 정의하고 세그먼트 구성
  • 월별 누적 이탈(Churn) 분석: 고객 생애주기와 리텐션 패턴 파악
  • 카이제곱 검정 및 t-test: 선호도 차이의 통계적 유의성 검증

📊 주요 성과:

  • 충성 고객(12.3%)이 전체 매출의 34% 기여
  • 충성 고객의 평균 지출이 일반 고객 대비 2.8배
  • CLV 기반 타겟팅 및 마케팅 예산 최적화 전략 제안

🎯 배운 점:

  • 고객 세분화: RFM 기반 데이터 기반 고객 분류 경험
  • 통계 검증: 가설 설정 → 검정 → 해석까지의 분석 프로세스 역량
  • 비즈니스 전략: 고객 가치 기반 마케팅 전략 도출 능력
