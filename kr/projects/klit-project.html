🎯 한국문학 번역 분석 플랫폼

프로젝트 기간: 2025.06 - 2025.07 (1개월)

👥 팀 규모: 5명

📌 핵심 스킬: LLM 기반 텍스트 분석 • 데이터 증강 • 클러스터링 & 페르소나 분석 • Streamlit 대시보드

📋 프로젝트 개요:

아마존 베스트셀러, IMDB 인기 지표 등 외부 데이터를 분석하여 해외 시장 성공 가능성이 높은 한국 소설을 식별하고, 글로벌 번역 시장 진출을 지원하는 데이터 기반 의사결정 플랫폼을 구축했습니다.

⚙️ 기술적 접근:

  • LLM(Gemini API) 활용: 장르 분류 및 마케팅 표현 카테고리 자동화
  • SMOTified-GAN & TVAE: 데이터 불균형 완화를 위한 합성 데이터 생성
  • 모델 비교/교차검증: CatBoost, 로지스틱 회귀 등 모델을 비교하여 최적 프레임워크 선정

📊 주요 성과:

  • F1-score 0.52 → 0.61 향상으로 예측 성능 개선
  • FUZZY TOPSIS 기반 복합 평가지표 개발 및 번역 우선순위 산정
  • SHAP 분석으로 모델 해석 가능성 확보 및 주요 변수 영향력 시각화

🎯 배운 점:

  • 텍스트 데이터 처리: LLM API를 활용한 대규모 텍스트 분류 자동화
  • 딥러닝 기반 데이터 증강: 불균형 데이터 환경에서의 생성 모델 적용 경험
  • 대시보드 개발: Streamlit 기반 의사결정 지원 도구 구축 및 배포
