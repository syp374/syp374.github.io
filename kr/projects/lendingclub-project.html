🎯 LendingClub 대출 미상환자 분류 모델

프로젝트 기간: 2025.04 - 2025.04 (3주)

👥 팀 규모: 5명

📌 핵심 스킬: 분류 모델링 • 클래스 불균형 처리 • 통계 분석

📋 프로젝트 개요:

미국 P2P 대출 플랫폼 LendingClub 데이터를 활용해 대출 미상환(부도) 예측 모델을 개발하고, 심각한 클래스 불균형 상황에서도 성능을 개선하여 리스크 관리 방안을 제안했습니다.

⚙️ 기술적 접근:

  • SMOTE 오버샘플링으로 불균형(정상:부도=9:1) 문제 완화
  • XGBoost, LightGBM 등 앙상블 기반 분류 모델 비교/선정
  • 150+ 변수에서 핵심 변수 선택 및 파생변수 생성

📊 주요 성과:

  • ROC-AUC 0.89 달성(베이스라인 대비 17% 개선)
  • 부도 탐지율 68% 확보 및 심사 개선안 제시
  • 손실 15~20% 절감 효과 기대

🎯 배운 점:

  • 성능 최적화: 피처 엔지니어링 및 하이퍼파라미터 튜닝 경험
  • 금융 리스크 모델링: 비용 기반 평가 관점에서의 실무 적용 고려
  • 고차원 데이터 처리: 핵심 변수 선별과 최적화 역량
