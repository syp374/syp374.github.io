🎯 Olist 브라질 이커머스 데이터 분석

프로젝트 기간: 2025.05 - 2025.05 (3주)

👥 팀 규모: 5명

📌 핵심 스킬: Tableau 대시보드 • 시계열 분석 • 마케팅 퍼널 분석

📋 프로젝트 개요:

브라질 B2B 이커머스 플랫폼 Olist의 판매/고객/물류 데이터를 분석하여 셀러 경험을 개선하고 신규 셀러 유치 전략을 제안했습니다.

⚙️ 기술적 접근:

  • 다중 이상치 탐지: Z-Score, IQR, Isolation Forest로 비정상 거래 패턴 탐지
  • 관계형 데이터 정합성 확보: 다대다 관계 중복을 정리하여 병합 품질 개선
  • 마케팅 퍼널 분석: 전환 병목을 찾아 개선 포인트 도출

📊 주요 성과:

  • Tableau 기반 실시간 KPI 모니터링 대시보드 구축
  • Sales/Quality/Logistics/Marketing 관점의 경쟁력 강화 전략 제안

🎯 배운 점:

  • 데이터 품질 관리: 복잡한 관계형 데이터에서 정합성 확보 경험
  • BI 구축: 실시간 모니터링 대시보드 설계/구현 역량
  • 퍼널 분석: 고객 여정 최적화를 위한 데이터 기반 인사이트 도출
